Nakama lua snippet for VSCode

# How to use it

Copy `lua.json` to `C:\Users\[Username]\AppData\Roaming\Code\User\snippets`

for `require` type `nk`

for `functions reference` type `nk.`

for `match` type `match`

for `match handler reference` type `match_`
